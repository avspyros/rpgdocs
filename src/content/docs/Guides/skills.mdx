### Using Skills

1. GM calls for an attempt at the skill and the player rolls **1d6**.
2. GM rolls himself if player should not know the result.
3. If result **equal to, or less than** the skill score then the action is successful.
4. For each point a **Specialist** puts in **Sneak attack** the damage multiplier is increased by **1**.
