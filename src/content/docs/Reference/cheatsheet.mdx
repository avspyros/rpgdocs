<Aside title="Actor:" type="caution">
  Character, PC, NPC or creature.
</Aside>

<CardGrid>
  <Card title="Surprised">
    - Roll a d6. 
    - If the result is 1-2, the target is surprised.

  </Card>
  <Card title="Sneak Attack">
    Attacking surprised (or unaware) actors.
    - For specialists with at least 2 points in the skill:
      - +2 attack bonus.
      - Multiply damage by number of Sneak points.
  </Card>
  <Card title="Attacking unaware foe from behind">
    - **Foe:** loses Dex & shield AC bonuses.
    - **Attacker:** +2 extra Attack bonus.
  </Card>
  <Card title="Parry">
    - Decide to parry at any point during the Round.
    - No other action is allowed.
    - +2 AC for the Round.
    - +4 AC for Elves & Dawrves.
  </Card>
  <Card title="Charge">
    - Use Running Move.  
    - x2 damage.
    - -2 AC penalty.
  
  </Card>
  <Card title="Press">
    - Only Fighters, Elves & Dwarves.
    - +2 attack.
    - -4 AC.

  </Card>
  <Card title="Defensive Fighting">
    - Only Fighters, Elves & Dwarves.
    - +2 AC.
    - -4 attack.

  </Card>
</CardGrid>
